# User story:
#     1. User must be able to view the menu.
#     2. User must be able to choose what they want.
#     3. User must be able to provide quantity for item.
#     4. User must be able to order additional items.
#     5. User should be able to exit ordering.
#     6. User should be able to view the bill.

# Algorithm/pseudocode
#     1. Display a welcome message with instructions for app.
#     2. Display the menu.
#     3. Get the menu choice from the user.
#         If the user choice was to order an item, take the user to place the order.
#         If the user wants to exit the app, provide an exit message with the bill. (Skip 4)
#     4. Get the price of that item and ask user for qty.
#     5. Calculate total price.
#     6. If user wants additional items, repeat steps from step 2.
